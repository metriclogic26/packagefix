node-fetch v2 to v3 Migration Guide — CommonJS to ESM

5 min read · Updated April 2026 · Safe version: node-fetch@3.3.2 or node-fetch@2.7.0

Breaking change — ESM only

node-fetch v3 is ESM only. If your project uses CommonJS (require()), you cannot upgrade to v3 without additional changes. You have two options: stay on node-fetch v2.7.0 (safe, CJS), or migrate to ESM.

Which version should you use?

Version Module system Status Use when
3.3.2 ESM only SAFE Project uses import/export
2.7.0 CommonJS SAFE Project uses require()
2.6.6 and below CommonJS VULNERABLE CVE-2022-0235, CVE-2020-15168

Option A — Stay on v2 (CommonJS projects)

Recommended for CJS projects
# Update to safe v2
npm install node-fetch@2.7.0

# Verify
npm list node-fetch
# Should show: node-fetch@2.7.0

# Your code stays exactly the same
const fetch = require('node-fetch');

Option B — Migrate to Node.js built-in fetch (best long-term)

Node.js 18+ includes a global fetch — no package needed. If you are on Node 18+, remove node-fetch entirely.

# Check your Node version
node --version
# If 18+, you can remove node-fetch

npm uninstall node-fetch

# Replace in your code:
# Before:
const fetch = require('node-fetch');

# After: nothing — fetch is global
const response = await fetch('https://api.example.com');

Option C — Migrate to node-fetch v3 (ESM projects)

Common errors when upgrading to v3

Error: require() of ES Module not supported

Error [ERR_REQUIRE_ESM]: require() of ES Module
/node_modules/node-fetch/src/index.js not supported.
node-fetch is loaded from
/app/node_modules/node-fetch/src/index.js
Did you mean to import node-fetch?

Cause: You upgraded to v3 but your project still uses require(). Fix: either downgrade to 2.7.0 or convert your project to ESM.

Error: Cannot use import statement in a module

SyntaxError: Cannot use import statement outside a module

Fix: Add "type": "module" to your package.json, or rename your files from .js to .mjs.

Before and after — v2 to v3

// v2 (CommonJS)
const fetch = require('node-fetch');
const res = await fetch('https://api.example.com');
const data = await res.json();

// v3 (ESM) — package.json must have "type": "module"
import fetch from 'node-fetch';
const res = await fetch('https://api.example.com');
const data = await res.json();
